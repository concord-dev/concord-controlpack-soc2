# concord-controlpack-soc2

SOC 2 control pack for [Concord](https://github.com/concord-dev/concord).

## Install

```sh
concord controlpack install ghcr.io/concord-dev/concord-controlpack-soc2:v0.1.0
```

## Evidence sources

This pack consumes evidence from the following Concord plugins:

github, okta, snyk
